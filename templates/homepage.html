The Wayland compositor
for every platform.

Wawona is a native, high-performance Wayland compositor for macOS, iOS, and Android. Built on a shared Rust core with direct Metal and Vulkan rendering — zero-copy, bare-metal speed, no VMs.

100% Free & Open Source

Wawona is and always will be free software under the MIT License. No paywalls, no telemetry, no vendor lock-in. Audit the code, fork it, contribute — it belongs to the community.

App Store Compliant

Designed from day one to meet Apple and Google app review guidelines. Available to everyone — download from the App Store or Play Store. No sideloading, no jailbreak required.

Extensible

Port any Linux application to macOS and utilize Wayland directly on the same computer. Wawona will support user plugins for extending functionality far beyond the barebones compositor.

Native Compositing

Runs underneath the host OS's native window management — Quartz on macOS, UIKit on iOS, Android's WindowManager on Android. Wayland surfaces become first-class native windows with full system integration.

// Wayland surface → native macOS window
let surface = compositor.create_surface();
let window = quartz.promote_to_nswindow(surface);
window.make_key_and_order_front();

Zero-Copy Rendering

Direct Metal and Vulkan pipelines with zero-copy IOSurface integration. Client buffers map directly to GPU textures — near-instant frame delivery at native refresh rates.

One Core, Three Platforms

A shared Rust library manages all compositor state, protocol dispatch, and IPC. Platform frontends connect via UniFFI and C FFI bindings — every platform runs the exact same compositor logic.

Waypipe + Remote Apps

Run remote Linux GUI applications over SSH and display them natively. Built-in Waypipe and Dropbear SSH on mobile — no separate SSH client needed.

// SSH into remote Linux host
waypipe ssh user@server
