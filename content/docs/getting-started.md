# Getting Started with Wawona

Wawona is designed to be easy to run on macOS. 

## Prerequisites

- macOS 12.0 or later
- A Wayland client or toolkit (like GTK or Qt)

## Installation

Currently, Wawona is in early development. You can build it from source:

```bash
git clone https://github.com/aspauldingcode/Wawona
cd Wawona
make
./wawona
```

Stay tuned for official releases!
